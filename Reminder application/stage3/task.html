Description

Although your app works, it still is not able to remind you of anything. For that to happen, you need to be able to fetch current time and be able to run a timer to track when a reminder should pop up.

Timer class will be used to implement the timer logic.

To implement timer logic, you will be using Timer class and its methods. It can be read about here.

For additional info, try this link.

You should take the time interval from the main screen and be able to pop up the reminder after the selected minutes have passed in real time.

Similar logic can be used for minimizing the window after few seconds automatically.

As for the window pop-up feature, you can research this link. 

 

So in this stage you will be completing the following objectives :

1. Implement window pop-up and disappearing when required.

2. Implement timer logic for the application and bind that to window pop-up feature.

 

At the end of this stage, you will have a complete standalone reminder application. Congratulations!

 

Objectives

  • To add actionlistener to “OK” button in Set Reminders window. This button should create new reminder, which should be instance of the Set reminder window. The reminder should be stored in the JList which we previously added to Reminder application window.
  • Each line in the JList should consists of the following text – “Reminder Text: “some text entered by user”; Delay: “    one of the options of set Delay comboBox – 30 by default”; Period: “one of the options of set Delay comboBox – 0 by default”
  • Clicking OK button data from Set Reminders window like text from Reminder text, time in seconds represented in Set delay comboBox and Set Period comboBox should be used to set timer task for each reminder. 
  • When set delay time ended Set Reminders window  with the instance of previously created reminder should appear on the screen and some sound (music or sound ) should continue for several seconds after this the reminder task activity should be stopped and reminder disappear from the screen. 

Very important that figures in Set Delay  comboBox and period Delay comboBox should follow in the following order:

For set delay – 30, 25, 15, 5;
For set Period – 0, 5, 10, 20;

This order will be used for testing purpose of your application. 

Note* - set Period equaled to 0 means that the reminder will appear only once. Other periods like 5, 10, 20 seconds means that Reminder window should be repeated every 5, 10, 20 seconds.
