Description

In this stage, you will make a small addition to your reminder app that will elevate its look.

You will be adding gifs/images to the reminder app to make it more appealing.

For this, you can use the Image class and ImageIcon class available in Java language.

Note : You can now add new gifs/images to each pre-programmed reminder to make your application a bit more fun.

 

Objectives

To add actionListeners to EDIT and DELETE buttons of the Reminder Application window. User can highlight each line  of JList , see example below: 

  • When one of the lines with the text of represented reminder stored in  JList is highlighted and DELETE button is clicked the reminder should be deleted from JList;
  •  When one of the lines with the text of represented reminder stored in  JList is highlighted and EDIT button is clicked the Set Reminder window representing the chosen reminder in the JList should appear and can be active for changing Reminder text or set Delay or set Period in the timer task. Clicking OK button the Set Reminder window should store new data.
