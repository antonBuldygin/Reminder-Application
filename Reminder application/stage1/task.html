Description

In this stage, you will create the basic skeleton for the reminder application. We will use Swing to create the GUI for our application.

In Swing, we create the window using JFrame. You should extend this class and make modifications to its size and placement on screen as per your requirements.

JFrame, which comes from the javax.swing package, creates the main window where we will place our menus, buttons, images, gifs etc.

The setSize() method is used to define a size for our window where the first parameter is the width of window and second parameter is its height.

To set location for the application window, we use the setLocation() method. It takes 2 parameters and places the application window at given location of our computer screen. 

By default, if you try to close the window using the red X button then it would just disappear from the screen and keep running in the background. To avoid this from happening, you need to set a default behavior whenever a window is being closed which can be done by calling the setDefaultCloseOperation() and passing "EXIT_ON_CLOSE" to it.

Below is an example of how to use this information in your code :

JFrame f = new JFrame("Your Title");
f.setSize(500, 300);
f.setDefaultCloseOperation(EXIT_ON_CLOSE);
f.setLocation(1250, 500);

Once you have a window set up , you can add buttons, dropdown menus as required. For this project, you can use JLabel to show text right next to dropdowns and JPanel to divide your GUI into multiple parts.

Other components list and use can be found here and here.  

Write a program that would resemble the basic look of the reminder application that you want to create. The application must be able to minimize and close completely on pressing the exit button.

Note: Find out how to disable the maximize button for the application window.

 

Objectives

Create a window and give it the Reminder Application title.  The window size should be 500*300. 

The window should contain the following components:

  • JButton with the AddReminder name.
  • JButton with the EditReminder name.
  • JButton with the DeleteReminder name.
  • JList with the List of Reminders name.
  • JScrollPane with the Scroll Pane name.

Enable the components and make them visible. The JScrollPane size shouls be 480*100. 

To set a name to an element, use the setName() method.
